## VPython
 + Open cmd
    win + r
 + Go to the the python36/Scripts path
     ```bash
     cd <your python36/Scripts path>
     ```
 + Install vpython
    ```bash
    pip3 install vpython
    ```
 + Come into error
     + First try upgrading pip
        ```bash
        python -m pip install --upgrade pip
        
        pip3 install vpython
        ```
     + Upgrade setuptools
        ```bash
        pip install --upgrade setuptools
        
        pip3 install vpython
        ```
